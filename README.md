### How I Built This
* **The Design:** I went with a minimalist dark theme, using **Tailwind CSS** to keep the styling lean and the "Phillip Matsolo" branding bold and clear.
* **The Engine:** I chose **Next.js and React** because I value speed. I wanted a site that didn't just look good, but felt instantaneous to the user.
* **The Foundation:** Built with **HTML5 and JavaScript** best practices to ensure the site is robust and easy to maintain.

### My Growth Process
Building this wasn't just about putting code on a screen; it was about learning how to present a professional brand. I spent time refining the **call-to-action (CTA)** buttons—like the "Get in touch" and "View GitHub" sections—to ensure that anyone visiting knows exactly how to reach out or see my work.
